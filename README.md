# 🤖 Portfolio Bot

Merhaba! Bu proje, Discord üzerinde çalışan bir **Portföy / Proje Yönetim Botu**dur.  
Bot sayesinde kullanıcılar kendi projelerini kaydedebilir, listeleyebilir, güncelleyebilir, silebilir ve projelerine beceriler ekleyebilir.

Bu bot, özellikle yazılım projelerini düzenli tutmak ve kişisel portföy oluşturmak için geliştirilmiştir. 🚀

---

## ✨ Özellikler

✅ Yeni proje ekleme  
✅ Projeleri listeleme  
✅ Proje bilgilerini güncelleme  
✅ Proje silme  
✅ Projelere beceri ekleme  
✅ Kullanıcıya özel proje kaydı  
✅ SQLite veri tabanı kullanımı  
✅ Hatalı komutlarda kullanıcıya uyarı verme  
✅ Modal pencere ile daha modern kullanıcı deneyimi  

---

## 🧠 Bot Komutları

| Komut | Açıklama |
|---|---|
| `!start` | Botu başlatır ve kullanıcıyı karşılar |
| `!info` | Kullanılabilir komutları gösterir |
| `!new_project` | Yeni proje ekleme işlemini başlatır |
| `!projects` | Kullanıcının kayıtlı projelerini listeler |
| `!update_projects` | Proje bilgilerini günceller |
| `!skills` | Seçilen projeye beceri ekler |
| `!delete` | Seçilen projeyi siler |
| `!set_description` | Proje açıklamasını günceller |
